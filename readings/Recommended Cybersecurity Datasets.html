Recommended Real Open Cybersecurity Datasets

1. CICIDS 2017Highly Recommended

Source: Canadian Institute for Cybersecurity (University of New Brunswick)

Use Case: Intrusion detection, anomaly detection, attack classification

Official Download Page: unb.ca/cic/datasets/ids-2017.html
Kaggle Version (easier access): kaggle.com/datasets/cicdataset/cicids2017

What It Includes:

  • Network flow features
  • Benign vs malicious traffic
  • Multiple attack types:
    • DDoS
    • Brute Force
    • Botnet
    • Web attacks
    • Port scans

2. UNSW-NB15 Dataset

Source: Australian Centre for Cyber Security
Official Page: research.unsw.edu.au/projects/unsw-nb15-dataset
Kaggle Version: kaggle.com/datasets/mrwellsdavid/unsw-nb15

Includes:

  • 49 network features
  • Labeled attack categories
  • Realistic synthetic traffic
  • Excellent for classification projects

3. NSL-KDD Dataset Classic IDS

Kaggle: kaggle.com/datasets/hassan06/nslkdd

Includes:

  • Network intrusion records
  • Labeled attacks
  • Simplified version of KDD Cup 1999 dataset
  • Good for beginners
  • Lightweight
  • Easy to clean and model

4. Phishing Websites Dataset

UCI Machine Learning Repository: archive.ics.uci.edu/ml/datasets/phishing+websites
  • Structured
  • Good for binary classification
  • Excellent intro to ML-based detection

5. Email Spam Dataset Cybersecurity

UCI Spam Dataset: archive.ics.uci.edu/ml/datasets/spambase
  • Spam detection
  • Feature-based classification
  • Good for probability/statistics application
